# Hostel Sync AI

AI-Powered Hostel Management Platform

Developed for WebForge – A Smart Campus Webathon 2026

## Team Information

**Team Name:** Codeverse

**Team Lead:** Tanish Jain

---

## Problem Statement

Most colleges and universities lack a centralized system for managing hostel operations. Room allocation, complaints, leave requests, visitor management, fee tracking, and communication are often handled manually, resulting in inefficiencies and delays.

---

## Features

### Student Portal
- Dashboard
- Room Details
- Complaint Submission
- Leave Requests
- QR Gate Pass

### Warden Portal
- Complaint Monitoring
- Leave Approval
- Occupancy Tracking

### Admin Portal
- User Management
- Room Allocation
- Analytics Dashboard

### AI Features
- AI Complaint Categorization
- AI Insights Dashboard
- Hostel Health Score
